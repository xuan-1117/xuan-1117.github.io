平台端口：19999

后台管理界面：<font style="color:rgb(44, 44, 44);">19999/manager  账号密码root</font>

虚拟机自动获取ip

sudo vim /etc/network/interfaces

<font style="color:rgba(0, 0, 0, 0.8);background-color:rgb(240, 243, 255);">添加以下内容：</font>

```plain
auto eth0
iface eth0 inet dhcp
```

<font style="color:rgba(0, 0, 0, 0.8);background-color:rgb(240, 243, 255);">重启服务：service networking restart</font>

<font style="color:rgba(0, 0, 0, 0.8);background-color:rgb(240, 243, 255);">手动请求 DHCP 分配：</font><font style="color:rgb(27, 106, 199);">sudo</font><font style="color:rgb(0, 0, 0);"> dhclient eth0</font>

查看镜像

```plain
docker images
```

<font style="color:rgb(79, 79, 79);">运行镜像</font>

```plain
docker run -d -p 8080:80 -it 镜像名 
```
